# agcli — LLM/Agent Reference

## What is agcli?
A Rust CLI and SDK for the Bittensor decentralized AI network. It lets you manage wallets, stake TAO, transfer tokens, register on subnets, set weights, query chain state, view Dynamic TAO pricing, and manage identities. All data commands support `--output json` and `--output csv` for machine-readable output. Compatible with Python bittensor-wallet keyfiles.

## Installation
```bash
cargo install --git https://github.com/unconst/agcli
```

## Core Commands

### Wallet
- `agcli wallet create --name NAME` — create new wallet (coldkey + hotkey)
- `agcli wallet list` — list wallets
- `agcli wallet show --all` — show all wallets with hotkeys
- `agcli wallet import --name NAME` — import from mnemonic
- `agcli wallet regen-coldkey` — regenerate coldkey from mnemonic
- `agcli wallet new-hotkey NAME` — create new hotkey
- Note: automatically reads Python bittensor-wallet keyfiles (NaCl SecretBox + JSON format)

### Balance & Transfer
- `agcli balance` — show your balance
- `agcli balance --address SS58` — show someone's balance
- `agcli transfer DEST_SS58 AMOUNT_TAO` — send TAO

### Staking
- `agcli stake add AMOUNT --netuid N` — stake TAO on subnet
- `agcli stake remove AMOUNT --netuid N` — unstake
- `agcli stake list` — show all stakes
- `agcli stake move AMOUNT --from SRC --to DST` — move between subnets
- `agcli stake swap AMOUNT --netuid N --from-hotkey HK1 --to-hotkey HK2` — swap between hotkeys
- `agcli stake unstake-all` — unstake all from hotkey
- `agcli stake claim-root --netuid N` — claim root dividends
- `agcli stake add-limit AMOUNT --netuid N --price P` — limit order stake
- `agcli stake childkey-take PCT --netuid N` — set childkey take
- `agcli stake set-children --netuid N --children "proportion:hotkey,..."` — set children

### Subnets
- `agcli subnet list` — list subnets with real names from DynamicInfo
- `agcli subnet show N` — show subnet details with Dynamic TAO pricing
- `agcli subnet metagraph N` — view metagraph
- `agcli subnet hyperparams N` — show hyperparameters
- `agcli subnet register` — create new subnet
- `agcli subnet register-neuron N` — register on subnet N (burn)
- `agcli subnet pow N --threads 8` — POW registration (multi-threaded)

### Weights
- `agcli weights set --netuid N "uid:weight,..."` — set weights
- `agcli weights commit --netuid N "uid:weight,..."` — commit-reveal (blake2 hash)
- `agcli weights reveal --netuid N "uid:weight,..." SALT` — reveal committed weights

### Delegates
- `agcli delegate list` — list all delegates
- `agcli delegate show --hotkey SS58` — show delegate info
- `agcli delegate decrease-take PCT` — decrease validator take
- `agcli delegate increase-take PCT` — increase validator take

### View
- `agcli view portfolio` — full stake portfolio with subnet names and prices
- `agcli view dynamic` — Dynamic TAO info (prices, pools, volumes, emissions)
- `agcli view network` — network overview (block, issuance, stake, emission)
- `agcli view neuron --netuid N UID` — full neuron detail

### Identity
- `agcli identity show SS58` — query on-chain identity (Registry pallet)
- `agcli identity set-subnet N --name "Name" --github "repo"` — set subnet identity

### Swap
- `agcli swap hotkey NEW_SS58` — swap hotkey
- `agcli swap coldkey NEW_SS58` — schedule coldkey swap

### Root
- `agcli root register` — register on root network
- `agcli root weights "netuid:weight,..."` — set root weights

### Live Mode
- `agcli --live view dynamic` — poll Dynamic TAO prices, show deltas (12s default)
- `agcli --live 30 subnet metagraph 1` — track neuron changes every 30s
- `agcli --live view portfolio` — watch your portfolio in real-time
- Delta tracking highlights price changes, pool movements, stake/emission shifts

### Event Subscription
- `agcli subscribe blocks` — watch finalized blocks
- `agcli subscribe events` — stream all chain events
- `agcli subscribe events staking` — filter staking events only
- `agcli subscribe events transfer` — filter balance transfer events
- Supports `--output json` for structured event streaming
- Filter categories: all, staking, registration, transfer, weights, subnet

### Configuration
- `agcli config show` — show current config
- `agcli config set KEY VALUE` — set: network, endpoint, wallet_dir, wallet, hotkey, output, proxy, live_interval
- `agcli config unset KEY` — remove a setting
- `agcli config path` — show config file path (~/.agcli/config.toml)
- Priority: CLI flags > env vars > config file > defaults

### Proxy
- `agcli --proxy SS58 <command>` — wrap any extrinsic through Proxy.proxy for delegated signing
- SDK: `client.proxy_call(&pair, "real_ss58", "Pallet", "call", fields).await?`

### Staking Wizard
- `agcli stake wizard` — interactive guided staking flow
- Shows balance, top subnets by pool, lets you pick subnet/amount, confirms before submitting

### Raw Calls (Advanced)
- SDK method: `client.submit_raw_call(&pair, "PalletName", "call_name", fields)`
- Enables calling any pallet (EVM, MEVShield, Contracts) without compile-time bindings

## Output Formats
Add `--output json` or `--output csv` to data queries:
```bash
agcli --output json subnet list
agcli --output csv view dynamic
agcli --output json view portfolio
agcli --output csv subnet metagraph 1
agcli --output json stake list
```

## SDK Usage
```rust
use agcli::{Client, Config, Wallet, Balance};

let client = Client::connect("wss://entrypoint-finney.opentensor.ai:443").await?;
let balance = client.get_balance_ss58("5Gx...").await?;
let subnets = client.get_all_subnets().await?;
let stakes = client.get_stake_for_coldkey("5Gx...").await?;
let metagraph = client.get_metagraph(1.into()).await?;
// Dynamic TAO pricing
let dynamic = client.get_all_dynamic_info().await?;
let sn1 = client.get_dynamic_info(1.into()).await?;
// Batch weight operations
client.batch_set_weights(&pair, &[1,2], &[weights1, weights2], &[0,0]).await?;
client.batch_commit_weights(&pair, &[1,2], &[hash1, hash2]).await?;
// Event subscription
agcli::events::subscribe_events(client.inner_client(), EventFilter::All, false).await?;
// Raw call to any pallet (EVM, MEVShield, etc.)
use subxt::dynamic::Value;
client.submit_raw_call(&pair, "EVM", "associate_evm_key", vec![...]).await?;
// Proxy call (delegate signing through proxy account)
client.proxy_call(&pair, "5RealAccount...", "SubtensorModule", "add_stake", vec![...]).await?;
// Load persistent config
let cfg = Config::load();
println!("Network: {:?}", cfg.network);
```

## Key Concepts
- TAO = native token, 1 TAO = 1e9 RAO
- Subnet = independent network with netuid (u16)
- Coldkey = main signing key (encrypted), Hotkey = automated key (plaintext)
- Weights = validators score miners, u16 values normalized to [0,1]
- Dynamic TAO = each subnet has alpha token, price determined by AMM pool
  - `tao_in` = TAO locked in subnet pool
  - `alpha_in` / `alpha_out` = alpha token balances
  - `price` = moving average price in TAO per alpha unit
- Root (SN0) = governs cross-subnet emissions
- Batch ops = set/commit/reveal weights across multiple subnets atomically
